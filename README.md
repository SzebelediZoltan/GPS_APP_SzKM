# GPS_APP_SzKM
A (NÉV) egy webalkalmazás, amely valós időben követi a jármű helyzetét, kijelzi az aktuális sebességet, a szakaszra érvényes sebességkorlátot, valamint figyelmeztet, ha a vezető túllépi azt. A térképen élő rétegként jelennek meg a közösségi jelzések (ideiglenes korlátozás, traffipont, útfelújítás)
